# Suit-Up API

A Node.js + Express + TypeScript backend for the SuitUp AI‑powered, made‑to‑measure suit e‑commerce MVP. Designed for scalability, robust logging, and Excel spec parsing.

---

## Prerequisites

- **Node.js** v18+
- **pnpm** v7+
- **Git** (with commitizen & commitlint)
- **Environment** variables defined in `.env` (see `.env.example`)

---

## Installation

```bash
# Clone and enter directory
git clone <repo-url> suit-up-api
cd suit-up-api

# Install dependencies
pnpm install

# Initialize Husky hooks
pnpm prepare
```
